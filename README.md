# luadist-i18n

This project is a metapackage to install [i18n.lua](https://github.com/kikito/i18n.lua) in [luadist](luadist.org)

# Install

```bash
$ git clone https://github.com/azukiapp/luadist-i18n
$ luadist install luadist-i18n
```
